Natour

Natour allows tourist to book their favourite tours

The backend was built with Node.js, and MongoDB was used for database storage

Tours can be booked via Stripe payment gateway, also users can review and rate a tour amidst other cool functionalities

Technologies:
  • - JavaScript
  • - Node js
  • - Pug
  • - MongoDB
  • - Heroku
  • - Stripe's Payment Gateway
  • - Mapbox
  • - Redis
  • - Docker
  • - Sendgrid
  • - Multer
